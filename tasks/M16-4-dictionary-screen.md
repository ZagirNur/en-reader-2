# Задача M16.4 — Экран «Словарь»

**Размер.** M (~2 дня)
**Зависимости.** M16.1 (tokens), M16.2 (sheet/tabbar), M16.3 (progression API).
**Что строится поверх.** M16.6 (тренировки берут слова из этого экрана).

---

## О проекте (контекст)

**en-reader** — веб-читалка. Словарь — одна из 4 вкладок таб-бара. Показывает все слова, которые пользователь переводил, с фильтрами по статусу (`new / learning / review / mastered`), статистикой и возможностью открыть карточку слова с примером и действиями.

---

## Что нужно сделать

Сверстать экран словаря согласно прототипу, подключить к API, добавить детальную шторку по клику.

---

## Что входит

### 1. View `renderDictionary()`

Структура:
```
┌─────────────────────────┐
│  Словарь       42 слов  │   h1 + counter
├─────────────────────────┤
│  [5]    [12]   [23]     │   stats card (review/active/mastered)
│  повт.  учу    выуч.    │
├─────────────────────────┤
│  [Все] [Повт.] [Учу]…   │   filter chips
├─────────────────────────┤
│  • ominous — зловещий   │   word card
│    "She whispered..."   │
│    The Great Gatsby · 3д│
│                 [учу]   │   status badge
│  • ...                  │
└─────────────────────────┘
```

### 2. Header

```html
<div class="dict-header">
  <h1>Словарь</h1>
  <span class="counter">${total} слов</span>
</div>
```

Стили: flex, justify-content: space-between, padding `18px 22px 10px`.

### 3. Stats card

3-колоночный grid с крупными цифрами:
- «На повтор сегодня» (22 px, 700, `var(--accent)`) + «Повт. / сегодня» (10 px `var(--ink-2)`).
- «Учу сейчас» (22 px 700 `var(--ink)`).
- «Выучено» (22 px 700 `var(--ink)`).

`font-variant-numeric: tabular-nums` на цифрах.

### 4. Filter chips

- `Все` + количество.
- `Повторить` + count.
- `Учу` + count.
- `Новые` + count.
- `Выучено` + count.

Active — `.chip.on`. Подставляется в запрос `/api/dictionary?status=<id>`. Горизонтальный скролл если не влезают.

### 5. Список слов

Для каждого слова — `.card` (p: 14 px, gap: 10):
- Левая колонка (flex: 1, min-width: 0):
  - headword 19 px 600 letter-spacing -0.01em + «— перевод» 13 px `var(--ink-2)`.
  - Пример: «"...example..."» 12 px `var(--ink-2)` line-height 1.4.
  - Meta: `source_book · N дн.` 10 px `var(--ink-2)` letter-spacing 0.02em.
- Справа — бейдж статуса.

Бейджи:
```js
const BADGE = {
  new:      {bg: 'var(--accent)', color: '#fff', label: 'новое'},
  learning: {bg: 'var(--soft)',   color: 'var(--ink)', label: 'учу'},
  review:   {bg: '#c9a253',       color: '#2d1a12', label: 'повторить'},
  mastered: {bg: '#d8e0c0',       color: '#2a3f14', label: 'выучено'},
};
// Для dark — mastered: {bg: '#2a3f24', color: '#c5d5a0'}
```

### 6. Пустое состояние

Если список пуст (с фильтром или без):
```
┌─────────────────────────┐
│  Здесь пока пусто       │
│  (ink-2, 13px, centered)│
└─────────────────────────┘
```

Если фильтр не all, а книг нет — «Нет слов со статусом <X>».

### 7. Клик по карточке → Word sheet

Через `openSheet()` из M16.2. Содержимое:
- Headword 34 px 600 + IPA/POS (italic 13 px `var(--ink-2)`) + close-btn 36×36.
- `.soft` card «Перевод» + перевод 18 px 500.
- Секция «Из книги · <source>» + пример с переведённым словом обёрнутым `<b style="color:var(--accent)">`.
- Actions: `btn primary` «Тренировать» → переход на `/learn/card?from=dict&lemma=<lemma>` · `btn ghost` «Удалить» → `DELETE /api/dictionary/{lemma}` + close sheet + toast «Удалено».

### 8. Интеграция с табами

В таб-баре (M16.2) `id='dict'` → рендер этого экрана. После открытия — `renderTabBar('dict', ...)`.

### 9. Тесты

- Фронт-юнит на формирование фильтра: клик по чипу меняет URL/state и делает новый запрос.
- Интеграция: seed 5 слов разных статусов, открыть «Словарь», проверить счётчики + фильтрацию.
- E2E (см. M15.6) не нужно отдельно; будет в общей резолюции.
